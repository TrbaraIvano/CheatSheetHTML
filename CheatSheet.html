Word Wide Web General

  • Die drei wichtigsten Technologien sind HTML, URL und HTTP.
  • Ressourcen werden über URLs (Uniform Resource Locators) adressiert.
  • Die allgemeinen Komponenten einer URL sind: Schema, Host, optionaler Port, Pfad, Query und Fragment.

Web Server

  • Ein Webserver ist Software oder Hardware, die Webinhalte bereitstellt.
  • Ein Webserver empfängt Anfragen von Clients und liefert passende Antworten wie HTML-Seiten oder Dateien.
  • Inhalte können als Dateien oder über Programme/Datenbanken auf dem Server gespeichert werden.
  • Inhalte von Webseiten werden mit HTML beschrieben.

Web Client

  • Ein Webclient ist ein Programm, das Inhalte von einem Webserver anfordert und darstellt.
  • Ein Webclient sendet Anfragen über HTTP und zeigt die empfangenen Daten an.
  • Ja, ein Webbrowser ist ein typischer Webclient.

Hello World

  • Das HTML head-Element enthält Metainformationen wie Titel, Zeichensatz oder Styles.
  • Die Deklaration !DOCTYPE html zeigt dem Browser, dass es sich um ein HTML5-Dokument handelt.
  • Die zwei obersten Elemente innerhalb von html sind head und body.
  • Text kann mit em hervorgehoben werden, wird meistens kursiv angezeigt.
  • Wichtiger Text kann mit strong markiert werden, wird meistens fett angezeigt.
